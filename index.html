Keypress Options

Key Function
Left Arrow Rotate model left about y axis (fast)
Right Arrow Rotate model right about y axis (fast)
Up Arrow Rotate model up about x axis (fast)
Down Arrow Rotate model down about x axis (fast)
e Halve (double) shininess exponent
i Rotate light source down (up) about x axis
j Rotate light source right (left) about y axis
p Toggle perspective/orthographic projection
r Restore defaults
x Rotate camera up (down) about x axis
y Rotate camera left (right) about y axis
z Rotate camera CCW (clockwise) about z axis
{"<"} Zoom in
{">"} Zoom out

Options in parentheses are selected with the Shift key.
